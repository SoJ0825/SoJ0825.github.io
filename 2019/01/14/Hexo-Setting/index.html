讓 Hexo 支援 markdown 語法

Hexo 設置

首先我要先謝謝 好想工作室 的前輩 Chris大大，本文大部分都是參考他的文章，大家可以去逛逛他的筆記，很多技術相關的文章可以參考。

Chris 筆記 - Hexo 研究筆記 1

預先安裝

  1. NodeJS => 會順便安裝 npm 套件管理系統
brew install node
  1. Hexo 主程式 => 透過 npm 安裝
npm install hexo-cli -g
  1. 建立一個新資料夾並安裝 hexo 基礎框架
hexo init blog
  1. 進入資料夾 cd blog，安裝相關套件
npm install
  1. 啟動本機 server(本機先查看結果)
hexo server

支援 markdown 語法

讓 Hexo 支援 markdown 語法，有一個最大好處，就是因為 HackMD 本身也是支援 markdown 語法，這樣在 HackMD 上的筆記，也可以快速轉換成我們部落格的貼文。

以下請參考 Chris 筆記

  1. 刪除原始渲染器
npm uninstall hexo-renderer-marked
  1. 安裝套件 hexo-renderer-markdown-it
npm install git+https://github.com/hexojs/hexo-renderer-markdown-it.git --save
  1. 安裝渲染器外掛(下面共11個)
npm install markdown-it-abbr markdown-it-container markdown-it-deflist markdown-it-emoji markdown-it-footnote markdown-it-imsize markdown-it-ins markdown-it-mark markdown-it-regexp markdown-it-sub markdown-it-sup --save
  1. HackMD 不用裝但 Hexo 要裝的套件
npm install markdown-it-checkbox --save
  1. 配置 Hexo 設定檔
# Markdown-it config
## Docs: https://github.com/celsomiranda/hexo-renderer-markdown-it/wiki
markdown:
  render:
    html: true # Doesn't escape HTML content so the tags will appear as html.
    # xhtmlOut: false # Parser will not produce XHTML compliant code.
    breaks: true # Parser produces `<br>` tags every time there is a line break in the source document.
    langPrefix: ''
    linkify: true # Returns text links as text.
    typographer: true # Substitution of common typographical elements will take place.
    quotes: '“”‘’' # "double" will be turned into “single”
    # 'single' will be turned into ‘single’
  plugins:
    - markdown-it-abbr
    - name: markdown-it-container
      options: success
    - name: markdown-it-container
      options: info
    - name: markdown-it-container
      options: warning
    - name: markdown-it-container
      options: danger
    - markdown-it-deflist
    - name: markdown-it-emoji
      options:
        shortcuts: {}
    - markdown-it-footnote
    - markdown-it-imsize
    - markdown-it-ins
    - markdown-it-katex
    - markdown-it-mark
    - markdown-it-regexp
    - markdown-it-sub
    - markdown-it-sup
    - markdown-it-checkbox
  anchors:
    level: 1 # Minimum level for ID creation. (Ex. h2 to h6)
    collisionSuffix: 'v' # A suffix that is prepended to the number given if the ID is repeated.
    permalink: true # If true, creates an anchor tag with a permalink besides the heading.
    permalinkClass: header-anchor # Class used for the permalink anchor tag.
    permalinkSymbol: '' # The symbol used to make the permalink.
  1. 安裝過濾器
    • 支援 流程圖
npm install hexo-filter-flowchart --save
  • UML 循序圖
npm install hexo-filter-sequence --save

數學表達式

Chris 筆記 - hexo加上數學式 MathJax

  1. 安裝套件
npm install markdown-it-katex --save
  1. 配置 hexo 設定檔
markdown:
  plugins:
    - markdown-it-katex
  1. 加入 CSS
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/KaTeX/0.5.1/katex.min.css">

其中 CSS 要加在 themes>>layout>>_partial>>head.ejs 的檔案裡，只要加在 <head> </head> 之間任意位置就可以

佈署到 github

  1. 安裝佈署套件
npm install hexo-deployer-git --save
  1. 配置 hexo 設定檔
deploy:
  type: git
  repo: <repository url>
  branch: [branch]
  message: [message]
  1. 上傳 github
hexo deploy -g
